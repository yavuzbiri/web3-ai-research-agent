# Web3 AI Research Agent

An AI-powered Web3 research and monitoring agent built with Shelby.

## Overview

This project is an automated AI agent that monitors new Web3 and crypto projects across multiple platforms and analyzes their activity using AI.

The goal is to help users quickly understand which projects are gaining traction and identify potential opportunities or risks.

## Features

- Collects data from Twitter/X, GitHub, and crypto news APIs
- Uses AI to analyze sentiment and project activity
- Generates summarized insights
- Sends alerts to Discord or Telegram

## Architecture

Data Sources
↓
API collectors
↓
AI analysis agent
↓
Alert system (Discord / Telegram)

## Tech Stack

- Python
- Node.js
- APIs (Twitter, News)
- AI models
- Shelby AI agent orchestration
